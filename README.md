# PredictStarNumberMod

## これは何
[BetterSongListに星予測機能をいれたMod](https://github.com/rakkyo150/BetterSongList-PredictStarNumber)の後継Modです。<br>
オリジナルのBetterSongListとともに使用してください。<br>
また、Standard以外のLawlessなどの譜面にも対応できるようになりました。<br>
今後はこちらを更新していきます。<br>
